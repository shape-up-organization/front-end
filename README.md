**The aesthetic Front-end**

#

### Getting Started

To run the project, first clone the repo, then follow the commands:

```bash
npm install
# or
yarn install
# or
pnpm install

npm run dev
# or
yarn dev
# or
pnpm dev
```

Now open [http://localhost:3000](http://localhost:3000) with your browser.
